# BlockPlane for Revit - Quick Start Guide

Get up and running with BlockPlane in 5 minutes!

## Installation (2 minutes)

1. **Download** `BlockPlane.Installer.msi`
2. **Double-click** to run installer
3. **Accept** license and click Install
4. **Launch** Revit 2024 or 2025

## First Use (3 minutes)

### Step 1: Open a Project
- Open an existing Revit project
- Or create a new project with some elements

### Step 2: Configure Settings
1. Click **BlockPlane** tab in ribbon
2. Click **Settings**
3. Set your **default region** (for pricing)
4. Click **Test Connection** to verify
5. Click **OK**

### Step 3: Analyze Your Project
1. Click **Carbon Calculator**
2. Wait for analysis (1-5 minutes)
3. Review results:
   - Total carbon footprint
   - Lifecycle breakdown
   - Top contributors
   - Optimization potential

### Step 4: Browse Materials
1. Click **Material Browser**
2. Try searching for "timber" or "concrete"
3. Filter by category or RIS score
4. Click any material to see details

### Step 5: Swap a Material (Optional)
1. Select some elements in your model
2. Click **Material Swap**
3. Review alternative materials
4. Choose a lower-carbon option
5. Click **Confirm Swap**

## What's Next?

- **Generate Reports**: Export PDF or CSV reports
- **Optimize Project**: Review top contributors and swap materials
- **Explore Database**: Browse thousands of sustainable materials
- **Read Full Guide**: Check out the complete User Guide
